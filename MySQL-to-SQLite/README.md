<h1>MySQL to SQLite</h1> 
	
MySQL to SQLite convertion script.

<h2>Description</h2>
If you need to convert a mysql database into a sqlite3 database. 

<h2>Usage</h2>
	<code>$ chmod +x mysql2sqlite.sh</code><br />
	<code>$ ./mysql2sqlite.sh databasename</code>
